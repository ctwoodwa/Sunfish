# sunfish-docs-change Routing

Navigate default stages for documentation-only changes.

**Acceleration:**
- Skip stage 01 (Discovery) if scope is clear
- Skip stage 02 (Architecture) — not applicable for docs
- Skip stage 03 (Package-design) — not applicable for docs
- Go straight from stage 00 to stage 05 (Implementation-plan)

**Main work happens in stage 06 (Build):**
- Write documentation
- Create/update kitchen-sink examples
- Verify all links and code examples work
- Review for clarity and accuracy

**Fast-track possible:** For simple updates (typo fixes, link fixes), bypass to stage 06 directly.
