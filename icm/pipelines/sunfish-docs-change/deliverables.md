# sunfish-docs-change Deliverables

## Core Deliverables

**Stage 06 (Build):**
- Updated markdown files in apps/docs
- Code examples in docs (if applicable)
- kitchen-sink demo updates (if applicable)
- JSDoc/XML code comments (if documenting API)

## Quality Checks

- [ ] All links work (no 404s)
- [ ] Code examples are correct and tested
- [ ] Markdown renders properly
- [ ] Examples match current API (not outdated)
- [ ] Tone is clear and helpful
- [ ] Navigation structure is clear

## Typical Docs Structure

- API reference: autogenerated or manually curated
- Usage guides: how to use features
- Examples: code snippets showing patterns
- Troubleshooting: common issues and solutions
