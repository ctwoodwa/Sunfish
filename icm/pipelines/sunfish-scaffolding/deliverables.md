# sunfish-scaffolding Deliverables

## Core Deliverables

**Stage 04 (Scaffolding):** Generator/template code in tooling/scaffolding-cli

**Stage 06 (Build):**
- Generated sample apps (Blazor and React, if applicable)
- Test results showing generated apps build and run
- Documentation of how to use new generators

**Stage 08 (Release):**
- Updated tooling/scaffolding-cli published
- CLI help/docs show new generators
- Example generated apps in documentation

## Quality Checks for Generated Code

- [ ] Generated apps build without errors
- [ ] Generated apps start without errors
- [ ] Generated code follows Sunfish conventions
- [ ] Generated code uses correct imports (foundation, ui-core, blocks)
- [ ] Generated examples are clear and representative
- [ ] Both Blazor and React targets work (if applicable)

## Template Requirements

- [ ] Template structure is clear and maintainable
- [ ] Generator code is parameterized (variables, options)
- [ ] Documentation explains how to modify templates
- [ ] Generated code can be customized by users
